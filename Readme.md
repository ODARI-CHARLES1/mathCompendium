# Math Compendium

A comprehensive collection of mathematical topics covering various disciplines.

## Contents

This compendium covers the following mathematical areas:

- **Algebra** - [`Alegbra/`](Alegbra)
- **Calculus** - [`Calculus/`](Calculus)
- **Complex Analysis** - [`ComplexAnalysis/`](ComplexAnalysis)
- **Numerical Methods** - [`NumericalMethods/`](NumericalMethods)
- **Ordinary Differential Equations (ODE)** - [`ODE/`](ODE)
- **Signal & Fourier Analysis** - [`SignalFourierAnalysis/`](SignalFourierAnalysis)
- **Statistics & Probability** - [`StatisticsProbability/`](StatisticsProbability)

## Overview

Each section contains dedicated documentation and materials for its respective mathematical domain. Browse the individual folders to access detailed content for each topic.

## Requirements

- A Markdown viewer for reading the documentation files
- Basic understanding of mathematical concepts
